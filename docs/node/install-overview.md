# Getting Help
There are multiple places node runners can find information and get assistance. Some are better than others depending on what one needs.
* [The xx Network Discord Channel](https://discord.gg/Y8pCkbK) - 24/7 technical support community with help often provided by seasoned node runners.
* [The xx Network Telegram Channel](https://t.me/xxnetwork) - Discussions are more general to xx Network but you're welcome to seek assistance there.
* [Email](mailto:nodes@xx.network) - Please be descriptive which helps the team understand the request up front. Replies can take up to 72 hours or more.
* [The xx Network Forum](https://forum.xx.network/) - A public forum which also has a section devoted to [How-Tos and Common Errors](https://forum.xx.network/c/everything-about-mainnet/how-to-and-freq-issues/34)
* [The xx Network Wiki](https://xxnetwork.wiki/Main_Page) - Documentation regarding the [Network Architecture](https://xxnetwork.wiki/Network_Architecture), [cMix Software Overview](https://xxnetwork.wiki/Xx_network_Software_Overview), and more.
# Linux Help
[Basic Linux Help](./references/basic-linux-help.md) is provided for users that may not be accustomed to the command-line or working from within a terminal. The section only provides help for topics one would need to complete the installation of the xx Network software. The content is not extensive so please make use of online search tools for Linux specific questions and help.
